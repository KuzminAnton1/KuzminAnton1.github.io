Методы и функции для работы со строками

Методы(функции) Описание
s.find('e') поиск символа'e' в строке s.
s.capitalize() возвращает копию строки s, в которой первый символ имеет верхний регистр, а все остальные символы имеют нижний регистр.
s.swapcase() возвращает копию строки s, в которой все символы, имеющие верхний регистр, преобразуются в символы нижнего регистра и наоборот.
s.title() возвращает копию строки s, в которой первый символ каждого слова переводится в верхний регистр.
s.lower() возвращает копию строки s, в которой все символы имеют нижний регистр.
s.upper() возвращает копию строки s, в которой все символы имеют верхний регистр.
s.count() (sub, start, end) считает количество непересекающихся вхождений подстроки sub в исходную строку s.
s.startswith() (suffix, start, end) определяет начинается ли исходная строка s подстрокой suffix
s.endswith() (suffix, start, end) определяет оканчивается ли исходная строка s подстрокой suffix
s.strip() возвращает копию строки s у которой удалены все пробелы стоящие в начале и конце строки.
s.replace(old, new) возвращает копию s со всеми вхождениями подстроки old, замененными на new
s.isalnum() определяет, состоит ли исходная строка из буквенно-цифровых символов.
s.isalpha() определяет, состоит ли исходная строка из буквенных символов.
s.isdigit() определяет, состоит ли исходная строка только из цифровых символов.
s.isspace() определяет, состоит ли исходная строка только из пробельных символов.
ord('символ') позволяет определить код некоторого символа в таблице символов Unicode.(num = ord('A') => 65)
chr('код символа') позволяет определить по коду символа сам символ.
bin() возвращает строку с двоичным представлением указанного целого числа.
oct() возвращает строку с восьмеричным представлением указанного целого числа.
hex() возвращает строку с шестнадцатеричным представлением указанного целого числа.

Форматирование строк

.format() - Python форматирует указанную строку и помещает их в строку на место заполнителей {}. Мы можем создавать сколько угодно заполнителей в строке:

age = 33
name = 'Anton'
profession = 'programmer'
txt = 'My name is {}, I am {}, I work as a {}'.format(name, age, profession)
print(txt)
# My name is Anton, I am 33, I work as a programmer

В Python 3.6 появилась новая разновидность строк — так называемые f-строки. Если поставить перед строкой префикс f, в заполнители можно будет включить код, например имя переменной.

first_name = 'Ivan'
last_name = 'Petrov'
age = 25
print(f'Hello, {first_name} {last_name}. You are {age}.')
# Hello, Ivan Petrov. You are 25.
