Модуль Random

Этот модуль предоставляет функции для генерации случайных чисел, букв и случайного выбора элементов последовательности (списка, строки и т.д.).

Функция randint() принимает два обязательных аргумента a и b и возвращает случайное целое число из отрезка [a;b]. Левая и правая граница a и b включаются в диапазон генерируемых случайных чисел.

Функция randrange() принимает такие же аргументы, что и функция range().

Функция random() возвращает случайное число с плавающей точкой (вещественное число). В функцию random() никаких аргументов не передается. Функция random() возвращает случайное число с плавающей точкой в диапазоне от 0.00.0 до 1.01.0 (исключая 1.01.0).

Функция uniform() тоже возвращает случайное число с плавающей точкой, но при этом она позволяет задавать диапазон для отбора значений.

Функция seed() в Python используется для инициализации случайных чисел. По умолчанию генератор случайных чисел использует текущее системное время. Если вы дважды используете одно и то же начальное значение, вы получите один и тот же результат, что означает случайное число дважды.

Пример:
import random
# здесь выставляется определенное значение
random.seed(10)
print(random.randit(1,20))
# 6
# при каждой последующей генерации, на выходе будет одно и то же значение
print(random.randit(1,20))
# 6

Функция shuffle() принимает список в качестве обязательного аргумента и перемешивает его случайным образом.

Функция choice() принимает список (строку) в качестве обязательного аргумента и возвращает один случайный элемент из переданного списка (строки).

Функция sample() принимает два обязательных аргумента: список (строку) и количество случайных элементов, а возвращает список случайных элементов в указанном количестве.

Модуль Re

Модуль re предоставляет полную поддержку регулярных выражений. Регулярные выражения — специальная последовательность символов, которая помогает сопоставлять или находить строки с использованием специализированного синтаксиса, содержащегося в шаблоне.

В таблицах ниже описаны все символы и комбинации символов для регулярных выражений, которые доступны в Python:

Оператор Описание
. Один любой символ, кроме новой строки \n.
? 0 или 1 вхождение шаблона слева
+ 1 и более вхождений шаблона слева
* 0 и более вхождений шаблона слева
\w Любая цифра или буква (\W — все, кроме буквы или цифры)
\d Любая цифра [0-9] (\D — все, кроме цифры)
\s Любой пробельный символ (\S — любой непробельный символ)
\b Граница слова
[..] Один из символов в скобках ([^..] — любой символ, кроме тех, что в скобках)
\ Экранирование специальных символов (\. означает точку или \+ — знак «плюс»)
^ и $ Начало и конец строки соответственно
{n,m} От n до m вхождений ({,m} — от 0 до m)
a|b Соответствует a или b
() Группирует выражение и возвращает найденный текст
\t, \n, \r Символ табуляции, новой строки и возврата каретки соответственно
Функции модуля Re:

re.compile(шаблон, repl, текст)- собирает регулярное выражение в отдельный объект, который может быть использован для поиска. Это также избавляет от переписывания одного и того же выражения. Фраза span=(3, 12) - показывает, где именно в строке программе удалось найти шаблон поиска.

re.match(pattern, string). Эта функция ищет по заданному шаблону в начале строки.

re.search(шаблон, текст). Эта функция похожа на match(), но ищет не только в начале строки.

re.findall(шаблон, текст). Эта функция возвращает список всех найденных совпадений. У метода findall() нет ограничений на поиск в начале или конце строки.

re.split(шаблон, текст, [maxsplit=0]). Эта функция разделяет строку по заданному шаблону.

re.sub(шаблон, repl, текст). Эта функция ищет шаблон в строке и заменяет его на указанную подстроку.

Модуль SQLite3

SQLite3 – это C библиотека, реализующая легковесную дисковую базу данных (БД), не требующую отдельного серверного процесса и позволяющую получить доступ к БД с использованием языка запросов SQL. Чтобы воспользоваться SQLite3 в Python необходимо импортировать модуль sqlite3, а затем создать объект подключения к БД. Объект подключения создается с помощью метода connect():
переменная = sqlite3.connect('путь к базе данных').

Создание таблицы в БД

После создания объекта соединения с базой данных нужно создать объект cursor. Он позволяет делать SQL-запросы к базе. Используем переменную access для хранения объекта. Функция execute отвечает за SQL-запрос. Команда IF NOT EXISTS поможет при попытке повторного подключения к базе данных. Запрос проверит, существует ли таблица. Если да — проверит, ничего ли не поменялось. Дальше создаем колонки в БД и их тип значения. В конце сохраняем изменения с помощью функции commit.
