Объявление и вызов функции

Итак, функция – отдельная, функционально независимая часть программы, выполняющая определенную задачу. Функции объявляются с помощью ключевого слова def (от слова define – определять). За ключевым словом def следуют название функции, круглые скобки (), и двоеточие :.

Первая строка объявления функции называется заголовком функции. Со следующей строки идет блок кода – тело функции. Это набор инструкций, составляющих одно целое и выполняющихся каждый раз, когда вызывается функция. Ниже приведенный фрагмент кода определяет функцию с именем show_message(). Тело ее состоит из одной инструкции, и вызов приведет к их исполнению. Для вызова функции пишут ее название и круглые скобки.

Параметры и аргументы функции

Аргумент – это любая порция данных, которая передается в функцию, когда функция вызывается. Параметр – это переменная, которая получает аргумент, переданный в функцию.

Локальные и глобальные переменные

Локальными называются переменные, объявленные внутри функции и доступные только ей самой. Программный код за пределами функции к ним доступа не имеет. Глобальными называются переменные, объявленные в основной программе и доступные как программе, так и всем ее функциям.
Рассмотрим функцию birds_russia(), которая выводит информацию о количестве птиц, обитающих в России. В теле функции мы создаем переменную birds, которой присваивается значение, равное 500. Такая переменная является локальной для функции birds_russia(). Всякий раз, когда переменной внутри функции присваивается значение, в результате создается локальная переменная. Она принадлежит функции, в которой создается, и к ней получает доступ только программный код этой функции. Термин локальная указывает на то, что переменная может использоваться лишь в этом месте — внутри функции, в которой создается. Если программный код одной функции попытается обратиться к локальной переменной, принадлежащей другой функции, произойдет ошибка.
